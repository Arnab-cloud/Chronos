# Privacy Policy

**Chronos** is an open-source, personal project. We value your privacy and aim to be transparent about how data is handled within the app.

## 1. Data Collection
Chronos is designed to function entirely offline. 
- **Personal Data:** We do not collect, store, or transmit any of your personal data (such as name, email, or location) to any external servers.
- **Task & Event Data:** All tasks, events, and schedules you create are stored locally on your device using a Room database. This data never leaves your device unless you manually perform a system backup (e.g., Google Drive backup).

## 2. Permissions
The app may request the following permissions to provide its core features:
- **Notifications:** Used to send reminders for your scheduled tasks and events.
- **Alarm/Exact Alarm:** Required to trigger reminders precisely at the scheduled time.

## 3. Third-Party Services
Chronos does not include any third-party analytics (like Firebase Analytics) or advertising SDKs.

## 4. Open Source
Since this is an open-source project, you can verify how your data is handled by reviewing the source code at [https://github.com/Arnab-cloud/Chronos](https://github.com/Arnab-cloud/Chronos).

## 5. Changes to This Policy
I may update this Privacy Policy from time to time. Any changes will be reflected by updating this file in the repository.
